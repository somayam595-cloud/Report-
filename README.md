# منصة تقارير البرامج المدرسية

نسخة ثابتة وآمنة مهيأة للنشر على **GitHub Pages**، وتعمل بالكامل داخل المتصفح دون خادم ودون مفاتيح API.

## الوظائف

- إنشاء عدد غير محدود من التقارير وحفظها تلقائياً داخل المتصفح.
- إنشاء مسودة ذكية محلية للأهداف والخطوات والنتائج والتوصيات.
- إضافة أربع صور للشواهد مع ضغط الصور تلقائياً.
- معاينة التقرير مباشرة بمقاس A4.
- الطباعة والتنزيل بصيغة PDF.
- تصدير نسخة احتياطية واستعادتها لاحقاً.
- استيراد التقارير المحفوظة من النسخة السابقة للموقع.
- تصميم عربي متجاوب للكمبيوتر والجوال.

## التشغيل على الكمبيوتر

يلزم تثبيت Node.js إصدار 20 أو أحدث، ثم تشغيل:

```bash
npm install
npm run dev
```

لفحص وبناء النسخة النهائية:

```bash
npm run build
```

## النشر على GitHub Pages

1. ارفعي **جميع محتويات هذا المجلد** إلى جذر المستودع.
2. افتحي المستودع في GitHub ثم:
   `Settings → Pages`.
3. من `Build and deployment` اختاري:
   `Source: GitHub Actions`.
4. انتظري اكتمال إجراء النشر من تبويب `Actions`.

ملف `vite.config.ts` يكتشف اسم المستودع تلقائياً أثناء GitHub Actions؛ لذلك تعمل مسارات الصور وملفات JavaScript حتى لو تغير اسم المستودع.

## ملاحظات مهمة

- البيانات تحفظ في `localStorage` داخل المتصفح المستخدم، وليست قاعدة بيانات سحابية.
- استخدمي زر **تصدير** للاحتفاظ بنسخة احتياطية من التقارير.
- لا تضعي أي مفتاح ذكاء اصطناعي داخل كود GitHub Pages، لأن ملفات الموقع عامة ويمكن لأي شخص قراءتها.
